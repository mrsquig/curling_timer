Timer Help/Tips

  • Use "Timer Adjustments" to add or remove time from the clock. This can be useful if the timer is started early/late, and can be done while the timer is running.
  • "Load server configuration profile" allows you to load pre-saved timer configurations. Selecting a profile will update the "Timer Options" form with the saved values.
  • After changing any options in the "Timer Options" form, be sure to click "Save Options" to apply the changes.
  • The main difference between the "League" and "Bonspiel" game types is that you define how many ends to play in League mode, while in Bonspiel mode the number of ends is automatically calculated based on the "Time to chime" value, when a warning is given to finish the current end and play one more.
  • Because of this, the "Time to chime" value should be set to a time that falls at the beginning of (or within) the desired second-to-last end (e.g. in a standard 8-end game, a value of 95 minutes would be 5 minutes into the 7th end and the timer would calculate the game includes 8 ends total).
  • IMPORTANT: The scheduler which starts the timer for regularly scheduled leagues will not function if the timer is in Bonspiel mode. Therefore, it is imperative that the timer is set back to League mode after the bonspiel is complete.

Please see the Timer Documentation for detailed instructions on using the timer.
